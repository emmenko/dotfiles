dotfiles
========

My dotfiles

## Installation

Install package managers

```bash
# Homebrew
ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

# Node
brew install node
```

```bash
# Set up custom OSX preferences, symlink dotfiles
./install.sh

# Update/Install brew formulae
./.brew

# Install npm packages
./.npm
```
